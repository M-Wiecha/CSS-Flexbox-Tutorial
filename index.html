Das CSS Flexbox-Labor

Flexbox richtet Elemente ganz einfach entlang einer Hauptachse aus. Schau dir an, wie sich die Boxen im grauen Container verteilen!

Vorschau: (display: flex;)
Box 1
Box 2
Box 3

Wichtige Flexbox-Befehle zum Testen:

1. Hauptachse ausrichten

Nutze justify-content im Code, um die Boxen horizontal zu verschieben:

Werte: flex-start, center, flex-end, space-between

2. Querachse ausrichten

Nutze align-items für die vertikale Ausrichtung im Container:

Werte: flex-start, center, flex-end, stretch

3. Richtung ändern

Mit flex-direction drehst du die Achse. Aus nebeneinander wird untereinander:

Werte: row, column
